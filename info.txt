Методы:
1) Поддерживается архивация ZIP, JAR
2) Поддерживается AES для шифрования

Входной файл:
1) Поддерживается 4 случая входного файла:
  а) Зашифрован и заархивирован
  б) Заархивирован и зашифрован
  в) Только зашифрован
  г) Только заархивирован

Выходной файл:
1) Поддерживается вывод в console, txt, json, xml.
2) Выходной файл может быть:
  а) Зашифрован и заархивирован
  б) Заархивирован и зашифрован
  в) Только зашифрован
  г) Только заархивирован
  д) Просто файлом
  
Интерфейс:
1) Для удобного использования разработан consoleUI

Тесты:
1) Тест для вычисления выражений
2) Тесты для чтения, записи txt (+ тест если файл не существует)
3) Тест для записи json
4) Тест для записи xml
